# Данные для проекта Toxic Comments NLP

Основной файл данных не хранится в публичном репозитории, потому что его размер составляет около `65 МБ`.

Скачать датасет можно по прямой ссылке:

- [https://code.s3.yandex.net/datasets/toxic_comments.csv](https://code.s3.yandex.net/datasets/toxic_comments.csv)

После скачивания:

1. помести файл в эту папку;
2. сохрани имя `toxic_comments.csv`;
3. открой ноутбук `identifying_toxic_comments_nlp.ipynb`.

Если удобнее через терминал:

```bash
curl -L https://code.s3.yandex.net/datasets/toxic_comments.csv -o data/toxic_comments.csv
```
